# Hello World

```kotlin
fun main() { 
    println("Hello World!")
}
```
바이브 코딩으로 지속적으로 개발을 진행해왔지만 ,

코드를 점검하지 못하고 개발중인 제품의 디테일이 떨어지는점을 체감하여

손코딩을 시작하게 되었다. 이번에 업로드 한 코드는 국민코드 Hello World 이다

프로그램의 진입점으로 사용되는 fun main() 함수부터 이를 출력해주는 println 까지

기본적인 내용들이지만 앞으로 아주 많이 사용하게 될것같다는 불길한 예감이 든다.
